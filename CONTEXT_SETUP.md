# PwC Fullstack Interview Preparation - Context

This project serves as a practice environment for the PwC Senior Associate Fullstack Developer interview.

**Tech Stack Overview:**
- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Recharts
- **Backend:** Node.js, Express, TypeScript
- **DevOps/Infrastructure:** Nginx, Docker, Kubernetes

**Primary Goals:**
1. Refresh core technical skills across the stack.
2. Complete hands-on coding assignments (`PRACTICE_ASSIGNMENT.md`).
3. Prepare for L1 technical screening with mock interviews and Q&A.

**Repositories:**
- `practice-fe`: Frontend React application.
- `practice-be`: Backend Node.js application.
